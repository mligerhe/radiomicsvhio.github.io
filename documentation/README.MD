# Prerequisite

1. Install [Go](https://golang.org/doc/install)
2. Install [Hugo](https://gohugo.io/getting-started/installing/)
3. Clone Theme 

```bash
$:- git clone https://github.com/zerostaticthemes/hugo-hero-theme.git themes/hugo-hero-theme
```
 
# Commands

## Build minify Html

```bash
$:- hugo --minify -d docs
```

## Runing local

```bash
$:- hugo server
```
